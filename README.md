# Client Dashboard - Final Project

## Project Overview
This is a full-stack Client Dashboard built using:
- React (Frontend)
- FastAPI (Backend)
- MySQL (Database)

---

## Setup Instructions

### Backend Setup

```bash
cd backend
python -m venv venv
venv\Scripts\activate
pip install -r requirements.txt
uvicorn main:app --reload
```

Backend will run on:
http://127.0.0.1:8000

---

### Frontend Setup

```bash
cd frontend
npm install
npm run dev
```

Frontend will run on:
http://localhost:5173
